# 📱 PokéDex Dinâmica

Este projeto é uma PokéDex interativa que consome a PokeAPI, desenvolvida durante a trilha de CSS da DIO, com diversas melhorias personalizadas.

## 🔗 Demonstração
Você pode visualizar o projeto na prática através do link abaixo:
👉 **[Clique aqui para acessar a PokéDex](https://jucianasoares.github.io/sitepokedex/)**

## 🚀 Funcionalidades Extras

🃏 Detalhamento dos Cards (Cartas)
Aprimorei a exibição de cada Pokémon com foco em design e clareza de informações:
Estilização Dinâmica: O fundo de cada carta muda de cor automaticamente de acordo com o tipo principal do Pokémon (ex: laranja para Fogo, azul para Água).
Hierarquia de Informações: Exibição clara do número de identificação (#ID), nome e os selos de tipos.
Imagens de Alta Qualidade: Configurei a integração para buscar as artes oficiais mais nítidas da PokéAPI.
Efeito Hover/Touch: Adicionei um leve feedback visual ao interagir com as cartas, tornando a navegação mais viva.

* **Busca em Tempo Real:** Filtre Pokémons pelo nome enquanto digita.
* **Filtro por Tipo:** Navegação global por categorias (Fogo, Água, Planta, etc).
* **Loading Customizado:** Animação de Pokébola girando durante o carregamento.
* **Mobile First:** Totalmente otimizado para navegação em dispositivos móveis.
* **Lista de pokémons:** Todos os pokémons de todas as gerações.
